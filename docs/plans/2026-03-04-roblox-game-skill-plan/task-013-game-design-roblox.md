## Description

Write a Roblox-specific game design reference covering core loops, retention mechanics, genre theory, player psychology, and what makes Roblox games succeed.

## What To Do

1. **Overview** — When to load (game design questions, retention concerns, genre selection)
2. **Core Loop Design** — Primary action loop, progression engine, meta-progression, how they interconnect. Examples from successful Roblox games
3. **Retention Mechanics** — Daily rewards, streaks, limited-time events, social features, collection completion, FOMO mechanics (used ethically)
4. **Genre Theory** — What makes each genre work on Roblox:
   - Simulators: collect → upgrade → unlock → prestige loop
   - Tycoons: build → earn → expand → automate
   - Obbys: skill progression, checkpoint satisfaction
   - RPGs: character growth, narrative, social
   - Horror: atmosphere, tension, surprise
   - Battle Royale: tension → action → reward
5. **Player Engagement** — First-time user experience (FTUE), tutorial design, early hook (first 30 seconds), social mechanics that drive return visits
6. **Difficulty Design** — Difficulty curves, skill-based vs time-based progression, avoiding frustration, rewarding mastery
7. **Social Systems** — Multiplayer interactions that drive engagement, guilds/groups, leaderboards, cooperative vs competitive
8. **Update Strategy** — Content cadence, seasonal events, community feedback loops, feature prioritization
9. **Best Practices** — Test with real players early, iterate on data, balance depth and accessibility, respect players' time
10. **Anti-Patterns** — Infinite grind with no variety, pay-to-win progression, neglecting mobile experience, no social features
