## Description

Write the revenue optimization workflow.

## What To Do

1. **Step 1: Current State** — MCP: scan for MarketplaceService usage, list all GamePasses and DevProducts. Offline: Ask user to list all monetization
2. **Step 2: GamePass Review** — Evaluate each GamePass: Is the value clear? Is pricing appropriate? Is it discoverable in-game?
3. **Step 3: DevProduct Review** — Evaluate consumables: Are they compelling? Priced for repeat purchase? Does ProcessReceipt handle correctly?
4. **Step 4: Missing Opportunities** — Suggest monetization the game is missing based on genre best practices (load game-design-roblox knowledge)
5. **Step 5: Pricing Analysis** — Compare prices against Roblox norms, evaluate Robux-to-value ratio, suggest price tiers
6. **Step 6: Premium Integration** — Check if Premium payouts are optimized, suggest Premium-exclusive benefits
7. **Step 7: Ad Integration** — Evaluate if Rewarded Video Ads would fit, suggest placement points
8. **Step 8: Ethical Review** — Flag any potentially predatory patterns, ensure value transparency, age-appropriate monetization
9. **Step 9: Recommendations Report** — Prioritized list of monetization improvements with estimated revenue impact
